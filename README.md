# toolkitty-website

Official [toolkitty](https://toolkitty.org) website.

## Requirements

* Ruby `3.2.0`
* Jekyll `3.10.0`
* Bundler `2.4.1`

## Development

```bash
# Install Ruby dependencies
bundle install

# Run development web server
jekyll serve

# If that fails, try this
bundle exec jekyll serve
```
